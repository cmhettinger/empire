# Empire Brand Standards

This document defines the official visual identity for the Empire platform. It serves as the canonical reference for colors, typography, and logo usage across applications, reports, websites, dashboards, and documentation.

---

# Brand Philosophy

Empire is designed to feel:

- Engineered over decorative
- Professional over flashy
- Timeless over trendy
- Premium through restraint
- Clear and highly readable

The visual identity should communicate confidence, precision, and long-term stability.

# Typography

Empire uses a simple three-font system.

---

## Cinzel

### Purpose

Display font used only for brand identity.

### Usage

- Empire wordmark
- Major cover titles
- Brand marks
- Very high-level document titles

### Avoid

Do not use Cinzel for:

- Body text
- Paragraphs
- Tables
- UI
- Documentation

Cinzel should remain distinctive through limited use.

---

## Acumin Pro

### Purpose

Primary user interface and document font.

### Usage

- Page titles
- Section headings
- Subheadings
- Body text
- Tables
- Captions
- Metadata
- Navigation
- Dashboards
- Reports
- Presentations
- Website content

### Recommended weights

- Light
- Regular
- Medium
- Semibold
- Bold

Acumin Pro should be used for nearly all written communication.

---

## Source Code Pro

### Purpose

Monospaced font for technical content.

### Usage

- Source code
- Terminal output
- SQL
- File paths
- Configuration examples
- Command line snippets

Do not use Source Code Pro for general body copy.

# Typography Hierarchy

| Element | Font | Weight |
|----------|------|--------|
| Logo | Cinzel | Bold |
| Cover Title | Cinzel | Regular / Bold |
| H1 | Acumin Pro | Semibold |
| H2 | Acumin Pro | Semibold |
| H3 | Acumin Pro | Medium |
| Body | Acumin Pro | Regular |
| Tables | Acumin Pro | Regular |
| Metadata | Acumin Pro | Light / Regular |
| Code | Source Code Pro | Regular |

---

# Logo Usage

Approved logo lockups:

- Icon
- Horizontal lockup
- Vertical lockup
- Wordmark

Approved logo colors:

- Black
- White
- Empire Primary Red

Do not:

- Stretch the logo
- Rotate the logo
- Apply effects or shadows
- Change typography
- Substitute colors outside the approved palette

---

# Guiding Principle

Cinzel establishes the Empire identity.

Acumin Pro communicates information.

Source Code Pro communicates implementation.

The overall visual language should remain clean, restrained, and consistent across all Empire properties.
